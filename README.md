# Setup

> [!NOTE]
> Change the terminal default profile to `Git Bash`

```bash
npm install
cd frontend
npm install
cd ..
cd backend
npm install
cd ..
```

Create `.env` file for each folder `frontend`, `backend`

# Command

Run frontend:

```bash
npm run fe
```

Run backend:

```bash
npm run be
```

Run both frontend and backend:

```bash
npm run dev
```

To terminate press shortcut `CTRL + C`
